About Project

Recognizing the potential of Shopify's booming POS system, I developed the Self Checkout for Shopify app, an innovative solution that merges the convenience of self-checkout kiosks with the robust functionality of Shopify.


"The only way to win is to learn faster than anyone else." - Eric Ries

This project involved:

  • Conceptualizing and developing a comprehensive self-checkout system that integrates seamlessly with Shopify.
  • Automating syncing for products, collections, discounts, and orders.
  • Supporting multiple checkout modes, including Scan 'n' Go, Pickup, and Delivery.
  • Integrating with Eflyn Digital Signage application.
  • Configuring various behaviors related to fulfillment, inventory deduction, and more.
  • Creating intuitive automated menus from Shopify collections and enabling store-wide search.
  • Supporting multiple checkouts on one kiosk.

The Self Checkout for Shopify app was built custom on the Kiosk Platform using Node.js and MongoDB and integrates with Shopify via multiple APIs. The success of this application affirms my belief in the value of merging powerful platforms to create seamless and convenient customer experiences.
